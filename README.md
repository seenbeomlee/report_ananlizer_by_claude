## 재사용 가능한 프롬프트 모음

### HWP / PDF → TXT 변환

다음 경로에 있는 [.hwp / .pdf] 파일을 파싱해서 .txt로 변환해줘.

**입력 경로:** [절대경로]  
**출력 경로:** [절대경로] (없으면 생성해줘)  
**대상 파일:** 파일명에 "[키워드]"가 포함된 .hwp 및 .pdf 파일  
**제외 파일:** `._`로 시작하는 macOS 메타데이터 파일은 제외  

**특이사항:**
- 파일명이 한글이라 macOS NFD 인코딩 이슈가 있을 수 있으니 `unicodedata.normalize('NFC', ...)` 처리 필요
- HWPX 포맷(ZIP 기반)이 있을 수 있으니 magic byte로 판별해서 처리 (`PK\x03\x04`면 ZIP)
