## 핵심 법령

### 1. 배리어프리법 (2006)
정식 명칭: 「高齢者、障害者等の移動等の円滑化の促進に関する法律」

**✅ 조항 번호 검증 완료 (2026.06.29)**

| 조항 | 내용 | 한국 연결 |
|------|------|----------|
| **본법 제14조 제1항** | 특별특정건축물을 정령이 정하는 규모 이상으로 건축할 때 건축물 이동 등 원활화 기준(建築物移動等円滑化基準)에 **적합하게 해야 한다** (의무) | 편의증진법 개정 또는 신법으로 동등 조항 신설 논거 |
| **본법 제14조 제2항** | 신축 특별특정건축물은 준공 후에도 기준 적합 유지 의무 | 사후 유지 의무 도입 논거 |
| **본법 제14조 제3항** | 지방공공단체는 조례로 特別特定建築物 범위 확대, 규모 기준 강화, BF 기준 부가 가능 | 지자체별 자율 강화 모델 |
| **정령(政令) 제9조** | 연면적 2,000㎡ 이상(공중화장실 50㎡ 이상)에 적용 | 규모 기준 설정 참조 |
| **본법 제17조** | 특정건축물 계획 인정 — **자율** (의무 아님) | 자율 인증의 한계 보완 논거 (한국 BF 자율 인증과 동일 성격) |
| 지자체 기본구상 조항 | 배리어프리 기본구상 수립 근거 — 본법 제25조 이후로 추정 (번호 추가 확인 필요) | 지자체별 BF 체육시설 계획 의무화 논거 |

> ⚠️ 기존 "제10조·제11조" 번호는 오류였음 — 의무 적합 조항은 **본법 제14조**. 제10조·제11조는 교통시설(旅客施設) 관련이거나 정령(政令) 조항으로 추정.
> 지자체 기본구상 조항 번호(제25조 이후)는 추가 확인 필요.

**특별특정건축물(特別特定建築物) 중 체육시설 해당 항목 (확인)**
- 一般公共の用に供される体育館 (일반 공중용 체육관) → **特別特定建築物** ✅
- 一般公共の用に供される水泳場 (일반 공중용 수영장) → **特別特定建築物** ✅
- → 2,000㎡ 이상 신축·증축·개축·용도변경 시 **본법 제14조 의무 적용**
- 민간 볼링장·오락시설도 특별특정건축물 포함 (운동시설 일반 → 특정건축물, 공공용 체육관·수영장 → 특별특정건축물)

**출처**: e-gov 법령검색 https://laws.e-gov.go.jp/law/418AC0000000091/ / 내각부(CAO) 배리어프리법 개요 자료 / kijunhou.com 특별특정건축물 해설

### 2. 스포츠기본법 (2011)
「スポーツ基本法」

| 조항 | 내용 |
|------|------|
| 제2조 제5항 | "모든 국민이 장애 유무에 관계없이 안전하고 쾌적하게 스포츠를 즐길 수 있는 환경을 정비한다" 명시 |

**✅ 한국 스포츠기본법(2021) 비교 완료 (2026.06.29)**
- 한국 스포츠기본법 제2조(기본이념): "개인이 스포츠 활동에서 차별받지 아니하도록" — 일반 비차별 원칙 선언
- **장애 유무에 관계없이 시설 환경 정비를 명시하는 조항 없음**
- 결론: 한국 스포츠기본법은 일본 スポーツ基本法 제2조5항보다 **명시성 낮음** → 법적 공백 논거 지지

## 보고서 활용 문장 (수정본)

> 일본은 2006년 「高齢者、障害者等の移動等の円滑化の促進に関する法律」(배리어프리법) **제14조 제1항**에 따라 연면적 2,000㎡ 이상의 신규 공공체육시설(일반 공중용 체육관·수영장)에 배리어프리 기준 적합을 의무화하고 있다. 이에 비해 한국은 편의증진법상 BF 인증이 자율적으로 운영되어 공공 실내체육시설의 실질 접근 가능 인증률이 13.6%에 그치고 있다.
