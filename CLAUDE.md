```
mcp__kordoc__parse_table      # 표 추출
mcp__kordoc__parse_form       # 양식 추출
```

### 이전 기수 보고서 분석 시
보고서 파일을 받으면 자동으로 다음을 분석한다:
1. 주제 유형 분류 (복지/행정/경제/환경/디지털 등)
2. 등수-보고서 특성 상관관계 (상위 팀 공통점, 하위 팀 취약점)
3. 목차 구성 방식 및 논리 흐름
4. 데이터·통계 활용 수준
5. 이해관계자 의견 반영 방식
6. 법령 신설 vs. 개정 선택 패턴
7. 예산 산출근거 구체성

### 주제 추천 시 출력 형식
```
## 추천 주제 [N]: [정책목표를 담은 구체적 제목]

**핵심 문제**: (현황의 어떤 문제를 해결하는가)
**정책 대안 방향**: ① ... ② ... ③ ...
**연계 법령**: (개정 또는 신설 대상 법령명)
**예산 항목**: (주요 사업비 항목)
**이전 기수 차별점**: (69·70기와 어떻게 다른가)
**상대평가 경쟁력**: (18팀 중 차별화 포인트)
**리스크**: (주제 선정 시 주의사항)
```

주제 선정 우선 조건: 123대 국정과제 연계 / 법령 개정·신설 명확성 / 예산 산출 가능성 / 이해관계자 인터뷰 접근성 / MECE 구조화 용이성

### 보고서 작성 지원 시
- 모든 조언에 "채점 기준 [항목명]에 해당합니다"를 명시
- 정책보고서: 경쟁 우위 관점으로 조언, 포괄적 대안 즉시 지적·구체화
- 법제보고서: 법령입안 실무기준 형식 준수, 입안 난이도 높이는 방향 권고
- 예산보고서: 단가×수량×횟수 방식 상세화, 1~5년 중장기 분석 틀 제공
- 이전 기수 분석 결과를 항상 근거로 활용
